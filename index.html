高堡奇人读后感

发表于 2019-02-09 | | 阅读次数:

高堡奇人的世界

假如二战是轴心国胜利了，德国和日本一起瓜分了世界了，那将会是什么样的世界呢？
在高堡奇人中，就是这样一个架空的世界。
菲利普迪克通过笔下的几个人物，古董店老板齐尔丹，日本驻美工商会会长田芥，手工厂工人弗林克，弗林克的前妻朱莉安娜等人物，他们的处境和走向，来向我们展现着这个世界。
《易经》在本书中成了这些人物决策，询问事情的前景的工具，阴阳，吉凶，好坏，书中的人物通过《易经》来投石问路，而他们的未来也如同《易经》一样阴晴不定，福祸相依。
神谕总是令人捉摸不透的，什么是好的，什么是坏的，书中的人物总会对卦辞感觉莫名其妙，在日本统治下的美国，何尝不是如此。日本人在这里虽然是统治者，但是对美国的一些历史物件，手工艺品却及其喜爱和崇拜，他们接受着美国的礼仪，穿着美国人的社交服饰，行走于美国街头，让自己融入到美国文化中。而美国人虽然是被统治者，但是也被友好的对待，没有任何苦大仇深的感觉。这种统治相比于德国的残暴的屠杀压迫，更显得柔和而又高效。
易经，美国文化，日本人与其说是统治，不如说是在学习，对于被统治的人民对其放权，给予充分的自由，不像德国人使用秘密警察，严格监视和控制，这种方式十分高明。他们甚至让自己融入了美国人民之中，没有去严格的控制阶级与舆论，也没有一厢情愿的灌输思想和文化，实实在在的“共荣”的感觉。这是个更好的世界吗？没有人知道。

菲利普迪克笔下的人物总是细腻，情感丰富，思想深远，而《高堡奇人》中的朱莉安娜可谓典型。

朱莉安娜

好的小说情节总是生动的，越是跌宕起伏，越让人过瘾，而推动着这些情节的人物，又会反过来被情节所影响，而产生了变化和成长。朱莉安娜和其他人物都在成长着，但她的成长毫无疑问是作者更加用心，更与众不同的。她是书中唯一见到高堡奇人这个人的人物，毫无疑问是更加接近《蝗虫之灾》这本书的真相的人物。
为什么是她呢？
从出场时的软弱，抑郁，如同书中的大部分人物一样，郁郁寡欢，依赖他人，《易经》是来自于她的前夫弗林克，而《蝗虫之灾》是来自于伪装成司机的德国秘密警察乔，她看起来没有自己的东西，她不知道自己想要什么，只是这样平静的活着，追求时尚，喜欢幻想，和大部分普普通通的女孩并无两样。
跟着乔去丹佛玩，逛街买衣服，得意忘形，直到最终在旅馆才发觉乔的目的其实是想利用她来刺杀《蝗虫之灾》的作者霍森，从刚开始的一味求饶，消极讨好，到最后的奋而反击，一击致命。
看到此时，总是让人觉得，朱莉安娜的行为不像一个普通的女孩。但是仔细想想，其实作者在塑造这个人物的时候，何尝不是在她身上埋藏着自己最殷切的希望。通过这个美丽而又普通，好学求索的美国女孩，来粉碎了德国人的阴谋。
虽然最后当女孩来到了霍森家中，发现《蝗虫之灾》并不是由霍森写就的，而是来自《易经》的神谕。而易经的指示也不过是另一个平行世界的展现。

真相和死亡一样可怕，但是比死亡更难发现

没有英雄人物的挣扎痛苦，朱莉安娜可以为了自己心中的美好而反抗德国人的残暴，也可以在获知真相的虚无之后重新满怀希望的投入生活之中，如同《易经》一般道法自然，无声无息。

《走向共和》观后感(一)

发表于 2019-02-05 | | 阅读次数:

走向共和是一部精良而又深刻的电视剧，里面塑造的人物并不是我们课本上的单薄而又脸谱化的人物。每个人都是那么复杂而又深刻。

李鸿章篇

开头就是一段慈禧太后的鹦鹉不说话了来引出李鸿章的篇幅。
刚开始一直不明白，走向共和和李鸿章有什么关系。后来才发现，共和的英文是Republic,也就是民主制，这一概念无疑是西方传进来的。李鸿章虽然明面上是一个保皇的清朝大臣，殆精竭虑的想使清朝重新复兴，作为北洋水师的总司令，守护着北京这一王朝首都，相比于张之洞的南洋水师，无疑对着清国更加至关重要。如同洋务运动的口号，“师夷长技以制夷”，李鸿章在清朝被西方海军用炮弹轰开国门之后，开始大力推崇着西方思想，虽然口号只是想引入西方技术，洋为中用。但是在大量的留学生外派，大量的西方文化传播和洋人与中国通商交流后，无疑为中国的共和土壤提供了基础。这种引入必然是多方位的，立体的，技术的引入无疑会带着思想，文化的碰撞。
所以现在看来，走向共和从李鸿章开始，无疑是最合适的。而在描述李鸿章的这段时期，张之洞这一洋务运动的另一领袖，翁同龢这一迂腐保皇派，袁世凯这一富有才华但两面三刀的墙头派，还有作为帝国的决策者慈禧还有光绪。各方面的势力互相角力，到最终北洋海战的一败涂地，也宣告着洋务运动的失败，李鸿章在签订了马关条约之后也渐渐的从历史的舞台上褪去。
走向共和也开始进入了戊戌变法这一篇章。
